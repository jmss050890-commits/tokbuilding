# TokFam Photo Upload Folder

You can upload directly from /tokfam using the TokFam Admin Uploader.

If you prefer manual file placement, use these file names:

- miari-01-birthday.jpg
- miari-02-outfit.jpg
- miari-03-family-hug.jpg
- miari-04-playtime.jpg
- miari-05-naptime.jpg
- miari-06-highlight.jpg

Accepted formats: .jpg, .jpeg, .png, .webp

Once files are added, the TokFam gallery at /tokfam will show them automatically.

Optional security:

- Set TOKFAM_ADMIN_KEY in your environment to require an admin key before uploads.
