## How To Use The Agents

1. Open the agent hub.
2. Choose the agent you want.
3. Type your question or use voice input if available.
4. Read the response.
5. Use `Play Welcome` or spoken replies if you want voice output.

## Voice Notes

Agent voices depend partly on the browser and device being used.

That means:

- the voice can sound different on different devices
- some browsers have better voice options than others
- the lab can prefer the correct kind of voice, but the browser still provides the final voice engine

If a voice sounds wrong:

1. refresh the page
2. test again with `Play Welcome`
3. try a different browser
4. report which agent sounds off

## KPA Standards

KPA means **Keep People Alive**.

In simple terms, that means SVL products should:

- protect people
- reduce harm
- support health
- strengthen families and community
- avoid careless or reckless design
- stay mission-first

If something does not support life, protection, or meaningful empowerment, it does not fit the standard.

## Troubleshooting

### If a page does not load

- refresh the browser
- check that the URL is correct
- return to the main lab domain and navigate again

### If an agent does not answer

- try sending the message one more time
- confirm your internet connection
- refresh the page

### If the voice sounds wrong

- use `Play Welcome` again
- try another browser
- note which agent and browser had the issue

### If TokHealth looks different than expected

That may be normal.

The SVL TokHealth subdomain currently forwards to the original mobile build, so the experience may look different from the local SVL interface.
